# ParcelManagement API

<h1>Clean Architecture</h1>

A lightweight, educational .NET 8 Web API built using Clean Architecture principles and Constructor Dependency Injection.
This project demonstrates how to design modular, maintainable backend systems using modern .NET practices.

<h2>Features</h2>

- Clean Architecture (Domain → Application → Infrastructure → API)
- Dependency Injection (Constructor Injection)
- RESTful Endpoints
- In‑Memory Repository for learning
- Swagger UI (OpenAPI)
- .http file for easy endpoint testing
- Beginner‑friendly and production‑inspired structure
